Descrição detalhada

Declaração das classes de teste para validar as entidades do sistema.

Este arquivo contém as classes responsáveis por testar as entidades definidas no sistema, incluindo Conta, Viagem, Destino, Atividade e Hospedagem. Os testes verificam os cenários de uso de cada entidade, validando os atributos associados.
