\begin{DoxyEnumerate}
\item {\bfseries{Apresentação (IA)}}\+: Reúne as funcionalidades para o usuário.
\item {\bfseries{Serviço (IS)}}\+: Lida com a lógica de negócio.

\end{DoxyEnumerate}

\begin{DoxyItemize}
\item {\bfseries{Reutilização de Código}}\+: A lógica de serviço pode ser usada por várias interfaces de apresentação.
\item {\bfseries{Testes Independentes}}\+: Cada camada pode ser testada isoladamente.
